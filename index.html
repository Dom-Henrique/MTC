Abaixo estão listadas algumas das tecnologias que irei utilizar para o desenvolvimento do Aspie AI

Python

Desenvolvida por Guido Van Rossum em 1991, Python possui sintaxe simples e flexibilidade com diversas áreas, como análise de dados, desenvolvimento Web e atualmente vem sendo cogitada como uma boa escolha para iniciantes em inteligência artificial.

Python será usado em conjunto com uma API do Google Gemini, que me permite treiná-la com os meus conhecimentos. Além disso, o framework Flask será fundamental para o funcionamento correto em nuvem.

Banco de dados

Ainda em decisão, o banco de dados possui a função de armazenar os dados (avá) que serão usados pela IA para ajudar os usuários. Esses dados serão tirados de livros e biografias de figuras influentes, como Napoleão, Dom Pedro I/IV e II, Simón Bolívar.

Além disso, o desenvolvedor irá registrar suas próprias experiências.
